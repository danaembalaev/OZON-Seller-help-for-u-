# Авто‑сайт: Ozon CSV -> «Основной список» (Excel под печать)

## Что это
Мини‑сайт: заходите по ссылке, загружаете CSV‑отчёт Ozon, скачиваете готовый Excel «Основной список».

Логика совпадает с финальным форматом:
- статус: **Ожидает отгрузки**
- верх: **по Артикулу** (сумма количества, пометки `8(2в одну, 3в одну)` только если в исходнике были строки >1)
- низ: только отправления, где **несколько артикулов** (без номеров), строка — `арт1 — 2; арт2 — 1; ...`
  и она стоит **в столбце A**, остальные столбцы пустые (A может «заходить» вправо)

## Запуск локально
```bash
pip install -r requirements.txt
uvicorn app:app --reload
```
Откройте: http://127.0.0.1:8000

## Быстрый деплой на Render (самое простое)
1) Загрузите папку проекта на GitHub
2) На Render: New -> **Blueprint** -> выберите репозиторий
3) Render сам прочитает `render.yaml` и поднимет сайт
